# Quickstart: 分散型配信情報共有ネットワーク(YP代替)

機能が end-to-end で動くことを検証する手順。契約の詳細は [contracts/](./contracts/)、
データ定義は [data-model.md](./data-model.md) を参照。

## 前提

- Windows 10/11 x64
- Rust stable ツールチェーン(`rustup`)— 開発時のみ。利用者は配布 exe のみで動作
- 検証用: PeerCastStation(配信側テスト)、任意の YP ブラウザ(index.txt 表示テスト)
- nostr リレー: ローカル検証はモックリレー(テスト内蔵)。実網検証は任意の公開リレー URL を 1 つ以上用意

## 1. ビルドとテスト

```powershell
cargo build --release          # 単一 exe: target\release\peca-p2p-yp.exe
cargo test                     # unit + contract + integration
cargo test --test cucumber     # Gherkin シナリオ(spec.md 対応。実装前は失敗すること)
```

期待結果: すべてのテストがパス(実装完了後)。Gherkin にはネガティブシナリオ
(過大ペイロード拒否・なりすまし検出・大量偽登録耐性)が含まれる。

## 2. 起動と初期設定

```powershell
.\target\release\peca-p2p-yp.exe
```

1. ブラウザで `http://127.0.0.1:7180/` を開く
2. **ペルソナ作成**: ペルソナ画面で新規作成(label は任意。ローカル専用)
3. **リレー登録**: 掲示板/SNS 等で入手したリレー URL 群をリレー画面に貼り付けて一括登録
   (既定リレーは同梱されない — research R10)

期待結果: `GET /api/v1/status` でリレー接続数 ≥ 1。

## 3. 掲載側の検証(US1 / SC-001)

1. PeerCastStation の YP 設定に `pcp://127.0.0.1:7146/` を追加
2. PeerCastStation で配信を開始(掲載 YP に本ソフトを指定)
3. UI の「掲載中」画面で当該チャンネルと掲載成功リレー数を確認

期待結果: 配信開始から 60 秒以内に、別ノード(手順 4)の一覧に表示される。
チャンネル詳細変更(曲情報等)が 60 秒以内に反映される。配信終了で `status=ended` が発行される。

## 4. 発見側の検証(US2 / SC-004)

別 PC(または別プロファイルの 2 プロセス目)で:

1. 本ソフトを起動し、同じリレー URL を登録
2. `http://127.0.0.1:7180/` の一覧に手順 3 のチャンネルが表示されることを確認(5 秒以内)
3. YP ブラウザの取得先に `http://127.0.0.1:7180/index.txt` を設定し、一覧表示と
   文字化けの有無を確認(Shift_JIS 既定 — research R5 のリスク解消ポイント)
4. 一覧のトラッカー情報(TIP)経由で PeerCastStation の視聴が開始できることを確認

## 5. 継続性の検証(US3 / SC-002)

1. リレーを 2 つ以上登録した状態で手順 3〜4 を成立させる
2. リレーのうち 1 つを無効化(または停止)する
3. 掲載・一覧が中断なく継続することを確認
4. すべてのリレーを無効化 → UI に到達不能バナーが出ること、再有効化で自動的に掲載再開することを確認

## 6. セキュリティシナリオの検証(spec セキュリティシナリオ)

自動テスト(cucumber + モックリレー)で実行:

- 16KB 超イベント → 破棄+`nostr_oversize` ログ(SC-007)
- 署名不正イベント → 一覧に出ない+`nostr_invalid_sig` ログ(SC-005)
- 大量偽チャンネル投入 → ミュート/リレー切離し/PoW 閾値で一覧の実用性維持
- `javascript:` 等のコンタクト URL → UI で警告フラグ(FR-012)

## 7. セットアップ時間の計測(SC-006)

初見ユーザーが「exe 入手 → 起動 → リレー貼り付け → 一覧閲覧」を 15 分以内に
完了できることを、手順書(README)に沿って計測する。
